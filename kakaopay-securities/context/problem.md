# 문제 정의 (카카오페이증권)

> **가중치 규칙**: 출제자 힌트 영상(S1, `sources.md`)은 출제 기업이 직접 준 힌트로 리서치 전체에서 최상위 가중치를 갖는다. 문제 선정·해석이 다른 출처와 충돌하면 S1이 우선한다.

## 출제자가 명시한 문제 (S1, 01:03–01:25)

초보 투자자가 매수·매도할 때 겪는 두 가지 고통:

1. **"내가 손해 보지 않았나?"** — 자신의 매매가 손해였는지 판단하지 못하는 불안.
2. **"매수·매도를 어떻게 해야 할지 모르겠는데, 어떻게 도움받을 수 있나?"** — 주문 실행 방법·판단에 대한 막막함.

"이런 부분에서 많은 분들이 괴로워한다는 생각이 들어서 이 문제를 냈습니다." — 문제 도메인은 출제자가 직접 확정해 줬다. 리서치는 이 고통의 실재를 공개 자료로 입증하고 구체화하는 데 집중한다.

## 출제자가 명시한 평가 기준 (S1, 01:26–02:00)

- 접할 수 있는 정보는 대부분 동일 → 수학적 최적점 찾기 문제가 **아니다**.
- 가장 중요한 것: 적절한 해답 + **사용자가 충분히 납득하고 안심하고 다음 행동을 실행할 수 있게 정보를 충분히 주는가, 궁금증을 해결하는가**.
- 뾰족한 정답보다 **해답을 만들어 가는 과정의 설계**를 본다 ("설득해 나가는 과정을 볼 것").

## 설계 제약/방향 (S1에서 도출)

- 형태: 앱이 아니라 **에이전트** — "의도를 말하면 실행해 주는 카카오페이증권의 에이전트" 비전(03:28–04:14)과 정합해야 함.
- 출제자 본인이 Codex 사용자(03:11) — Codex 플러그인 형태는 출제 의도와 자연스럽게 맞음.
- 제출물 전반(README·로그·question.md)에 "문제 이해 → 논리적 분해 → AI 해법 → 스스로 증명" 흐름을 드러낼 것(솔버 인재상, 02:00–02:21).

## 1. 컨텍스트 (출처: sources.md)

카카오페이증권은 "초보 투자자도 쉽게 투자하고 자산을 운용하게 돕는" 포지션의 증권사다(S1 00:18–00:33, S5). 대표 서비스 '주식 모으기'는 출시 2년 만에 이용자 160만 명을 넘었고 56%가 '매일 모으기'를 선택했다(S4) — 핵심 고객이 소액·장기 성향의 초보 투자자임을 보여준다. 한편 국내 개인투자자, 특히 신규 투자자는 매매 판단에서 체계적으로 실패한다: 2020년 신규 진입자의 60%가 손실(S3), 거래회전율 1,600%·평균 보유 9.67일의 과잉거래, 이익 주식을 손실 주식보다 2배 이상 먼저 파는 처분효과(S2).

## 2. 문제 후보

1. **"내가 손해 봤나?"를 스스로 판단 못 함** — 초보 투자자가 매매 후 수수료·세금·환율·평균단가·기회비용까지 반영한 실질 손익과 그 이유를 이해하지 못해 불안해함. 근거: S1(01:03–01:25, 출제자가 직접 명시), S2·S3(처분효과·신규투자자 60% 손실 — 손익 판단 실패의 통계적 실재).
2. **"매수·매도를 어떻게 해야 하나?"를 모름** — 주문 유형·호가·타이밍·수량 등 실행 방법 앞에서 막히고, 정보는 많지만 자기 상황에 맞게 납득시켜 주는 것이 없음. 근거: S1(01:03–01:25), S2(과잉거래·짧은 보유 — 근거 없는 실행의 결과).
3. 상담사 지원 시스템 고도화(콜/채팅 상담 AI) — S1(00:34–01:02)에서 이미 사내에 구축했다고 언급 → **회사가 이미 푼 문제이므로 제외.**

## 3. Problem Statement (문항 2 초안 뼈대)

**초보 투자자는 매매 전후에 "이 거래가 손해인지, 다음에 무엇을 어떻게 해야 하는지"를 납득할 수준으로 설명받지 못해, 처분효과·과잉거래 같은 행태 오류로 시장수익률을 밑도는 손실을 본다.** 선정 이유: (a) 출제자가 영상에서 직접 지목한 문제(S1 — 최상위 가중치), (b) 통계로 실재가 입증됨(S2·S3), (c) 회사의 핵심 고객층(160만 초보 투자자, S4)과 정확히 겹침, (d) 후보 3은 이미 해결됨.

## 4. Opportunity Sizing (대략치, 출처 필수)

- 회사 내 직접 대상: '주식 모으기' 이용자 160만 명(S4) — 전원이 초보·소액 투자자 성격.
- 시장 전체: 2020년 신규 개인투자자 중 60% 손실(S3), 개인투자자 표본 20.4만 명 기준 과잉거래·처분효과가 일반적(S2) → "납득시키는 매매 설명"의 수요는 초보 투자자 모수 전체에 걸침.

## 5. → 플러그인 결정 (문항 1 초안 뼈대, 한 문장)

**초보 투자자의 계좌·거래 내역과 매매 의도를 입력받아, "손해인지"(수수료·세금·단가 반영 실질 손익 + 왜 그런지)와 "어떻게 할지"(주문 실행 단계)를 근거·과정을 드러내며 사용자가 납득할 때까지 설명하는 '매매 설득 에이전트' Codex 플러그인** — 뾰족한 정답이 아니라 설득 과정의 설계를 평가한다는 출제 기준(S1 01:26–02:00)에 정면으로 맞춘다.

> 사내 업무 번역(~2주 분량): 현재 상담사가 콜/채팅으로 건별 응대하는 "이 거래 손해인가요 / 주문 어떻게 하나요" 유형 문의를, 근거를 갖춘 설명 생성으로 대체·보조.
