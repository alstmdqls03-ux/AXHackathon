# step-00 — 출제자 의도 정리 (최상위 가중치 자산화)

먼저 `../orchestration/prompts/RULES.md`의 공통 규칙을 적용하라. 이 스텝만 수행하고 정지한다.

## 목적

힌트 영상(출제 기업이 직접 준 자료)에서 출제자의 의도를 구조화해, 이후 모든 스텝이 인용할 최상위 가중치 문서를 만든다.

## 입력

- `context/hint-video-transcript.md`, `context/hint-video.md` (없으면 CLAUDE.md의 "첫 작업" 절차로 먼저 생성)
- 이전 지시로 이미 만든 리서치 파일(problem.md·sources.md 등)이 있어도 이 스텝에서는 손대지 않는다 — step-01·02에서 흡수한다.

## 작업

영상에서 다음을 추출하고, 각 항목에 `[mm:ss]` 타임스탬프 인용을 붙인다:

1. 출제자가 반복·강조한 문제의식과 가치
2. 언급된 구체적 업무·병목·페인포인트
3. 원하는 인재상과 AI 활용 방식 (평가 관점 시사)
4. 명시적 발언(사실)과 나의 해석(추정)을 구분해 표기

## 산출물

- `context/intent.md` — 최상단에 **"출제 기업 직접 힌트 — 리서치 전체에서 최상위 가중치, 타 출처와 충돌 시 우선"** 을 명시. 출처(영상 URL)와 수집일 기재.

## 게이트

- [ ] 모든 항목에 타임스탬프 근거
- [ ] 발언(사실) / 해석(추정) 구분 표기
- [ ] intent.md에 최상위 가중치 선언 + 출처 URL + 수집일

## 종료

`context/JOURNAL.md`에 규격 보고 후 정지.
