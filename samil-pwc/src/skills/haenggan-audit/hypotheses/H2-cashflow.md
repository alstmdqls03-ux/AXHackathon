# H2 — 현금흐름 이상 (이익의 행간에서 유동성 위험이 자란다)

## 가설

손익은 흑자를 유지하는데 영업현금흐름·운전자본의 행간(회수 지연, 특정 거래처 편중, 기말 밀어내기)에 유동성 위험이 누적되고 있다.

## 필요 데이터

- 필수: `monthly_series.csv` (월별 매출·영업이익·영업현금흐름·매출채권·재고)
- 보조: `ar_aging.csv` (거래처별 채권 연령 — 없으면 편중 분석만 생략)

## 체커 신호 해석

| check | 신호의 의미 | 단독으로 결론? |
|---|---|---|
| `profit_cf_divergence` | 이익-현금 괴리 지속 — 이익의 질 문제 | 아니오 (원인 규명 필요) |
| `dso_trend` | 회수기간 악화 — 괴리의 유력한 원인 | 아니오 |
| `quarter_end_spike` | 분기말 매출 몰아치기(밀어내기) 패턴 | 아니오 |
| `ar_concentration` | 특정 거래처 편중 + 장기 연체 — 회수 위험의 소재 | 아니오 |

## 추론 절차

1. 괴리(divergence)를 **원인 신호와 연결**한다: DSO 추세·채권 급증(시산표 매출채권 YoY 신호가 H1에서 넘어올 수 있음)·재고 추이 중 무엇이 현금을 잠식하는지 분해.
2. 밀어내기 신호와 결합한다: 분기말 스파이크 매출이 **회수로 이어졌는지**(다음 달 회수액·채권 연령) 확인 — 스파이크 직후 채권 급증이면 매출의 질 의심.
3. 거래처 수준으로 내려간다: 편중 거래처의 연체 구조(61일+ 비중)로 회수 위험을 특정.
4. 지속 시나리오를 계산한다: 현 추세(월 CF)로 몇 개월 내 자금 압박인지, 차입금 추이(시산표)와 함께 제시.

## 판단 기준 (지식)

- 이익의 질: 영업이익이 흑자여도 영업CF가 음수로 지속되면 발생주의 이익이 현금으로 회수되지 않는 것 — 매출채권·재고에 이익이 잠긴 상태.
- DSO(회수기간) = 매출채권 ÷ 최근 3개월 평균 매출 × 30. 급격한 상승은 회수 정책 완화·부실 징후·밀어내기 중 하나.
- 분기말 스파이크 + 회수 부진 + 특정 거래처 편중의 3결합은 **채널 스터핑(밀어내기)** 전형 패턴.
- 계속기업 관점: 유동성 위험은 감사보고서 강조사항(계속기업 불확실성)으로 이어지는 선행 신호.

## 기각(오탐) 조건

- 계절성 사업(연말 성수기 등)으로 스파이크가 설명되고 **전년 동월도 유사 패턴**이면 기각 — 단 전년 데이터가 없으면 '계절성 미확인'을 한계로 명시하고 확신도를 낮춘다.
- CF 음수가 일시적 대형 지출(설비 투자성 선급 등)로 설명되면 기각 — 전표·시산표에서 반대 증거 탐색.
- 채권 편중 거래처가 신용 우량(연체 없음)이면 편중 자체만으로는 '주의' 수준으로 강등.

## 리포트 필수 요소

월별 수치 표(이익 vs CF) / DSO 계산식과 전후 값 / 편중 거래처명·연체 버킷 인용 / 위험 시나리오(몇 개월 내 압박) / 권고(회수 정책·여신 한도·밀어내기 여부 내부 확인) / 확신도와 한계(전년 비교 부재 등).
